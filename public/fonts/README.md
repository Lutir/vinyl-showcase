# Retro Fonts for Vinyl Collection Showcase

This directory contains the custom fonts for the 1970s retro aesthetic.

## Required Fonts

### 1. Display Font (Cooper Black style)

- **Purpose**: Headings and titles
- **Style**: Chunky, bold, 70s display font
- **Recommended**: Cooper Black, Chunk Five, or similar
- **File**: `display.woff2`

### 2. Body Font (Avant Garde style)

- **Purpose**: Body text and descriptions
- **Style**: Geometric sans-serif
- **Recommended**: ITC Avant Garde Gothic, Futura, or similar
- **File**: `body.woff2`

### 3. Accent Font (Brush Script style)

- **Purpose**: Handwritten descriptions on vinyl cards
- **Style**: Casual script/handwritten
- **Recommended**: Brush Script, Pacifico, or similar
- **File**: `accent.woff2`

## Font Sources

You can obtain these fonts from:

- [Google Fonts](https://fonts.google.com/) - Free, open-source fonts
- [Font Squirrel](https://www.fontsquirrel.com/) - Free fonts for commercial use
- [Adobe Fonts](https://fonts.adobe.com/) - If you have Adobe Creative Cloud

## Suggested Free Alternatives

- **Display**: "Righteous" or "Bungee" from Google Fonts
- **Body**: "Montserrat" or "Raleway" from Google Fonts
- **Accent**: "Pacifico" or "Caveat" from Google Fonts

## Installation

1. Download the font files in WOFF2 format
2. Place them in this directory with the names:
   - `display.woff2`
   - `body.woff2`
   - `accent.woff2`
3. The fonts will be automatically loaded via `@font-face` rules in globals.css
